# evoNN_UTTT: The Neuroevolution Algorithm Trained on Ultimate Tic-Tac-Toe

This project is an attempt to create a problem-agnostic, efficient, and performant neuroevolution library. It is tested and developed, initially, for the board game of Ultimate Tic-Tac-Toe, as it is a non-trivial, non-differentiable, complete-information problem.

**The current version is broken and in the middle of experimental feature implementation; do not attempt to run it.**

**GitHub Pages Devlog: https://stefanciutac.github.io/stefanciutac_devlog/**

The devlog linked above contains posts discussing system architecture and the rationale behind the design decisions I have made, as well as discussing the project's next steps. The features which will be implemented soon are listed at the bottom of each post.

### Notes:
- **The project is currently a work in progress, and far from a first stable release**
- Although the board game of Ultimate Tic-Tac-Toe is used to develop and benchmark the alpha versions and first implementation of the system, the resulting library aims to be problem-agnostic

### System Prerequisites:
- The Eigen3 library must be installed
